平板智能时钟 Tablet Smart Clock

适合 Lenovo TAB E8 这类旧安卓平板的小工具：大字翻页时钟、实时天气、番茄钟、音乐控制、待办事项、快捷入口、主题背景、低亮和常亮模式。

GitHub 发布名：tablet-smart-clock-dashboard。README 里包含主页、翻页时钟、音乐控制台和番茄钟截图，方便用户直接判断用途。

推荐用法：
1. 能安装 APK 时，安装 ReviveBoard.apk。
2. 如果 APK 安装失败，把 ReviveBoard.html 拷到平板内部存储，用浏览器打开。
3. 平板长期插电使用时，建议打开系统的护眼模式或把屏幕亮度调低。

旧平板更适合固定任务：
床头钟、厨房菜谱屏、学习番茄钟、家庭留言板、老人常用网页入口、电子相册、局域网控制面板。

不要指望它流畅运行新版微信、QQ、短视频或大型浏览器标签页。把它变成一个专用设备，会比继续当主力平板舒服很多。
